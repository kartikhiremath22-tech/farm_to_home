HOW TO ADD YOUR OWN PRODUCT PHOTOS:

1. Find/take a photo of the crop (e.g. tomato.jpg)
2. Name it EXACTLY matching the crop name in lowercase, with underscores for spaces:
   - "Tomatoes" -> tomato.jpg or tomatoes.jpg (it matches if "tomato" is found in name)
   - "Sweet Corn" -> corn.jpg (matches "corn")
3. Put the file in this folder: static/images/
4. Supported formats: .jpg, .jpeg, .png, .webp

If no photo is found for a product, a nice emoji is shown automatically instead -
so the site always looks good even before you add photos.

Already-matched crop names (just add a photo file with these words in the name):
tomato, potato, broccoli, mango, onion, carrot, banana, apple,
rice, wheat, spinach, chilli/chili, cabbage, cucumber, corn,
garlic, lemon, orange, grape
